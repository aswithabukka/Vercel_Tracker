How to set up Firebase sync — step by step
  1. Go to console.firebase.google.com (sign in with Google)
  2. Click Add project → name it prep-tracker → skip Google Analytics → Create
  3. On the project home page, click the web icon </> to register a web app. Name it tracker → Register app
  4. You'll see a firebaseConfig block with apiKey, authDomain, etc. Copy just the inner object (the part inside the curly braces, including the braces themselves)
  5. In the left sidebar: Build → Realtime Database → Create Database → pick a location → Start in test mode → Enable
  6. Come back here and paste the config below. Click Save.
  7. After it works, lock down the database rules — see "Security after setup" below.
Security after setup (do this once it works)
  1. Firebase console → Realtime Database → Rules tab
  2. Replace the rules with this (replace the tracker ID with whatever you used): 
    {
  "rules": {
    "trackers": {
      "$id": {
        ".read": true,
        ".write": "$id === 'aswitha-prep-2026'"
      }
    }
  }
}
  3. Click Publish
  4. This means: anyone with the link can READ, but only your specific tracker ID can WRITE. Husband sees but can't edit.
