# Aswitha Prep Tracker (Vercel)

## Deploy to Vercel

1. Go to vercel.com, sign in with GitHub or email
2. Click "Add New Project"
3. Either:
   - Drag-drop this whole folder onto Vercel's deployment page, OR
   - Push to GitHub, then import the repo in Vercel
4. Vercel gives you a URL like `prep-tracker-xxx.vercel.app`
5. Done.
